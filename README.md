# deta-drive-proxy

### Syntax

```bash
project_id=c0m8f10j
drive_name=Drive
filename=abc/hello.txt
API_KEY=c0m8f10j_uvveYTnAqUmVWCZ6oeCiJa9hoRPiSjif
curl https://public.deta.dev/$project_id/$drive_name/$filename?key=$API_KEY
```

A key is required on the first request.
